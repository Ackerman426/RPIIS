# Лабараторная работа №1
### Цель:
**Научиться верстать при помощью LaTeX**

### Основные команды:
* `\par` - абзац
* `\vspace{N}` - оступ на N
* `\begin{itemize} ... \end{itemize}` - список без нумерации
* `\newpage` - новая страница
* `\nohyphens` - убрать отступ в начале(убрать абзац)
